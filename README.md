# mersge
Merge tool for command line

## Run
```mersge <file>```

## Usage
- ```Arrow Up``` - move up
- ```Arrow Down``` - move down
- ```L``` - accept local
- ```R``` - accept incoming
- ```W``` - write file
- ```Q``` - quit
 
## Build
```cargo build```
